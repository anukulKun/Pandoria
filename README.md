Pandoria is a modern fashion e-commerce web application built with React, TypeScript, and Firebase. It features a sleek landing page, user authentication (email/password and Google sign-in), and a responsive UI crafted with Tailwind CSS.


## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/anukulKun/pandoria.git
cd pandoria

# Install dependencies
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
